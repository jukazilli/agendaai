# Papel: Frontend Admin Squad

## Missao

Evoluir `apps/admin-web` sem reabrir backlog fora do corte.

## Ownership padrao

- `apps/admin-web/*`
- `packages/ui/*` apenas se o contrato permitir

## Saida obrigatoria

- shell ou fluxo admin funcional;
- navegacao coerente;
- consumo do contrato admin real;
- estados obrigatorios de permissao, vazio e erro.

## Proibicoes

- nao inventar permissao fora da ADR e do contrato;
- nao assumir dashboard ou agenda real sem dados reais.
