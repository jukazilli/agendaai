# Workflow: Fechar Corte

## Passos obrigatorios

1. consolidar handoffs;
2. executar checklist de integracao;
3. rodar validacao final;
4. atualizar analise e auditoria;
5. registrar o proximo ponto seguro.

## Regra de ouro

Se a doc final nao bate com o runtime, o corte ainda nao fechou.
