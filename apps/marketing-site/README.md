# AgendaAI Marketing Site

Esta app agora roda em `Astro`, alinhada com a stack oficial documentada em
`docs/02_fundacoes/00_fundacao_de_produto_e_stack.md`.

## Objetivo

- materializar o wordmark de forma visual e consultavel;
- dar uma referencia web local para MCP, Playwright e futuras telas do projeto;
- servir como semente da futura superficie publica institucional;
- preservar uma linguagem visual propria de AgendaAI.

## Como abrir localmente

Execute:

```powershell
pnpm --dir .\apps\marketing-site dev
```

Ou:

```powershell
.\scripts\serve-brand-guide.ps1
```

URL local esperada:

`http://127.0.0.1:4173`
