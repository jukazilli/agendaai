# Brand Assets

Arquivos desta pasta materializam o wordmark canonico de AgendaAI e alimentam a pagina web local do brand guide.

- `agendaai-wordmark.svg`: lockup primario com barras em ink, vetor ascendente em teal e wordmark `AgendaAI` em ink
- `agendaai-wordmark-reverse.svg`: lockup reverso para barras escuras e navegacao sobre fundo profundo
- `agendaai-wordmark-mono.svg`: versao monocromatica em ink para superficies tecnicas
- `agendaai-symbol.svg`: simbolo reduzido para favicon, avatar e shell compacto
